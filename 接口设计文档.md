 # 货柜仓储管理系统 - 接口设计文档（V1）
 
 ---
 
 ## 一、设计规范
 
 | 项目 | 规范 |
 |---|---|
 | 基础路径 | `/api/v1` |
 | 请求格式 | JSON（`Content-Type: application/json`） |
 | 响应格式 | JSON 统一结构 |
 | 认证方式 | 暂不考虑（第一期无登录） |
 | 分页参数 | `page`（页码，默认1）、`pageSize`（每页条数，默认10） |
 | 排序参数 | `sortBy`（排序字段）、`sortOrder`（asc/desc） |
 
 **统一响应结构**：
 
 ```json
 {
   "code": 200,
   "message": "success",
   "data": {}
 }
 ```
 
 **分页响应结构**：
 
 ```json
 {
   "code": 200,
   "message": "success",
   "data": {
     "list": [],
     "total": 100,
     "page": 1,
     "pageSize": 10
   }
 }
 ```

 ### 9. 贩卖机商品管理
 
 | 方法 | 路径 | 说明 |
 |---|---|---|
 | GET | `/machine-product/list` | 机器商品列表（机器详情页） |
 | POST | `/machine-product` | 为机器添加商品 |
 | PUT | `/machine-product/{id}` | 编辑机器商品（调整配货数量/售卖价格） |
 | DELETE | `/machine-product/{id}` | 移除机器商品 |
 | GET | `/machine-product/{machineId}/restock-suggest` | 补货建议 |
 | POST | `/machine-product/{machineId}/sync-stock` | 手动同步库存（调用贩卖机接口） |
 
 #### 9.1 机器商品列表
 
 **请求**：
 ```
 GET /api/v1/machine-product/list?machineId=1&page=1&pageSize=50
 ```
 
 **响应**：
 ```json
 {
   "code": 200,
   "message": "success",
   "data": {
     "list": [
       {
         "id": 1,
         "machineId": 1,
         "productId": 1,
         "productName": "可口可乐",
         "specName": "24个/箱",
         "brandName": "可口可乐",
         "sellPrice": 3.50,
         "stockQuantity": 20,
         "allocationQuantity": 50,
         "needRestock": true,
         "restockQuantity": 30,
         "stockPercent": 40.00
       }
     ],
     "total": 15,
     "summary": {
       "totalStock": 300,
       "totalAllocation": 600,
       "stockPercent": 50.00
     }
   }
 }
 ```
 
 **说明**：
 - `needRestock`：是否需要配货（系统自动计算）
 - `restockQuantity`：补货数量 = allocationQuantity - stockQuantity（仅 needRestock=true 时有效）
 - `stockPercent`：库存消耗百分比 = stockQuantity / allocationQuantity * 100
 - `summary`：全机器汇总
 
 #### 9.2 为机器添加商品
 
 **请求**：
 ```json
 POST /api/v1/machine-product
 {
   "machineId": 1,
   "productId": 1,
   "sellPrice": 3.50,
   "allocationQuantity": 50
 }
 ```
 
 **响应**：
 ```json
 { "code": 200, "message": "success", "data": { "id": 1 } }
 ```
 
 **错误码**：
 | code | message |
 |---|---|
 | 4004 | 该机器已存在此商品 |
 
 #### 9.3 编辑机器商品（调整配货数量）
 
 **请求**：
 ```json
 PUT /api/v1/machine-product/1
 {
   "sellPrice": 4.00,
   "allocationQuantity": 60
 }
 ```
 
 **响应**：
 ```json
 { "code": 200, "message": "success", "data": null }
 ```
 
 #### 9.4 补货建议
 
 **请求**：
 ```
 GET /api/v1/machine-product/1/restock-suggest
 ```
 
 **响应**：
 ```json
 {
   "code": 200,
   "message": "success",
   "data": {
     "machineId": 1,
     "machineCode": "VM-001",
     "items": [
       {
         "productId": 1,
         "productName": "可口可乐",
         "specName": "24个/箱",
         "brandName": "可口可乐",
         "stockQuantity": 20,
         "allocationQuantity": 50,
         "restockQuantity": 30
       },
       {
         "productId": 3,
         "productName": "薯片",
         "specName": "12袋/箱",
         "brandName": "乐事",
         "stockQuantity": 5,
         "allocationQuantity": 30,
         "restockQuantity": 25
       }
     ]
   }
 }
 ```
 
 **说明**：返回所有 `stockQuantity < allocationQuantity × 阈值百分比` 的商品，`restockQuantity` 严格按公式计算。

 #### 13.3 新建配货单（从补货建议带入）
 
 **请求**：
 ```json
 POST /api/v1/delivery-order
 {
   "machineId": 1,
   "restockerId": 1,
   "deliveryPersonId": 2,
   "vehicleId": 1,
   "items": [
     { "productId": 1, "quantity": 30 },
     { "productId": 3, "quantity": 25 }
   ]
 }
 ```
 
 **说明**：`items` 初始值从补货建议接口获取，配货人员可在此调整 `quantity`。
 
 **响应**：
 ```json
 { "code": 200, "message": "success", "data": { "id": 1, "orderNo": "DO20260712001" } }
 ```
 
 #### 13.4 更新配货（配货完成）
 
 **请求**：
 ```
 PUT /api/v1/delivery-order/1/complete-stock
 ```
 
 **说明**：自动记录 `completedTime` 为当前时间，状态更新为「配货完成」。
 
 **响应**：
 ```json
 { "code": 200, "message": "配货完成", "data": null }
 ```
 
 **错误码**：
 | code | message |
 |---|---|
 | 4005 | 当前状态不允许此操作 |
 
 #### 13.5 开始送货
 
 **请求**：
 ```
 PUT /api/v1/delivery-order/1/start-delivery
 ```
 
 **说明**：自动记录 `deliveryStartTime` 为当前时间，状态更新为「送货中」。
 
 **响应**：
 ```json
 { "code": 200, "message": "开始送货", "data": null }
 ```
 
 #### 13.6 完成配送（自动更新贩卖机库存）
 
 **请求**：
 ```
 PUT /api/v1/delivery-order/1/complete-delivery
 ```
 
 **说明**：
 1. 自动记录 `deliveryEndTime` 为当前时间
 2. 遍历配货明细，按 `quantity` 累加到 `machine_product.stock_quantity`
 3. 状态更新为「送货完成」
 
 **响应**：
 ```json
 { "code": 200, "message": "配送完成，贩卖机库存已更新", "data": null }
 ```
 
 **错误码**：
 | code | message |
 |---|---|
 | 4005 | 当前状态不允许此操作 |
